# 📉 Customer Churn Analysis Dashboard 🚀

## 🚀 Overview
Developed an interactive **Customer Churn Analysis Dashboard** using **Power BI, SQL, and Python** to analyze customer behavior, identify churn drivers, and measure revenue impact.  

Performed data cleaning using Python, conducted business analysis using SQL, and built dynamic visualizations in Power BI to support data-driven decision-making.

---

## 🛠️ Tech Stack
- **Power BI** – Dashboard development & data visualization  
- **SQL** – Business queries & churn analysis  
- **Python (Pandas)** – Data cleaning & preprocessing  

---

## ⚙️ Data Handling & Analysis
- Cleaned and processed raw telecom dataset using **Python**  
- Handled missing values and ensured data consistency  
- Performed **SQL-based analysis** to:
  - Calculate churn rate  
  - Identify high-risk customer segments  
  - Analyze revenue loss due to churn  
- Connected processed data to **Power BI** for visualization  

---

## 📊 Key Features
- Built **2 interactive dashboard pages**:
  - Customer Churn Overview  
  - Churn Driver Analysis  

- Developed KPIs:
  - Total Customers  
  - Churn Rate  
  - Revenue at Risk  
  - High-Risk Customers  

- Implemented dynamic filters:
  - Contract Type  
  - Internet Service  

- Analyzed churn across:
  - Contract Type  
  - Payment Method  
  - Internet Service  
  - Customer Tenure  

---

## 📈 Impact & Insights
- Identified overall churn rate of **~26.6%**  
- Detected a **high-risk segment** (Month-to-month + Electronic check)  
- Found high-risk customers have **~53.7% churn rate (3× higher than others)**  
- Discovered **~26% of customers contribute to over 50% of churn-related revenue loss**  

### Key Observations:
- Month-to-month contracts have the highest churn  
- Electronic check users churn the most  
- Fiber optic customers show higher churn behavior  
- New customers (0–1 year tenure) are most likely to churn  

---

## 📌 Business Value
- Helps identify **high-risk customer segments**  
- Enables targeted **retention strategies**  
- Supports **revenue loss reduction initiatives**  
- Improves **customer lifecycle management**  
- Provides actionable insights for **data-driven decision-making**
